# Screenshots Directory

Place your application screenshots here.

## Recommended Screenshots:

1. **login.png** - Login/Register page
2. **user-dashboard.png** - User dashboard with sweets list
3. **cart.png** - Shopping cart page
4. **payment.png** - Payment page
5. **admin-dashboard.png** - Admin inventory management
6. **admin-orders.png** - Admin orders view

## How to Add Screenshots:

1. Take screenshots of your running application
2. Save them in this `screenshots/` folder
3. Use the filenames listed above (or update the paths in README.md)
4. The screenshots will automatically appear in the main README.md

## Screenshot Tips:

- Use clear, high-quality images
- Show the key features of each page
- Ensure text is readable
- Recommended size: 1200x800 pixels or similar
